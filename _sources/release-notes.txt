=============
Release Notes
=============

.. note::

  This information is drawn from the GitHub MediaElch project changelog: |br|
  https://github.com/Komet/MediaElch/blob/master/changelog.md


2.4.3 (*tbd*)
***********************

Features
^^^^^^^^^^^^^^^^^^^^^^^

 - Use HTTPS for scraping (`#371 <https://github.com/Komet/MediaElch/issues/371>`_)
 - Set UI language in ````advancedsettings.xml```` (`#411 <https://github.com/Komet/MediaElch/issues/411>`_)
 - Add Kino.de scraper (`#370 <https://github.com/Komet/MediaElch/issues/370>`_)
 - Load *all* tags from IMDb (`#469 <https://github.com/Komet/MediaElch/issues/469>`_)
 - Add more audio and video codecs (`#524 <https://github.com/Komet/MediaElch/issues/524>`_, `#530 <https://github.com/Komet/MediaElch/issues/530>`_)
 - Remove deprecated Cinefacts (`#370 <https://github.com/Komet/MediaElch/issues/370>`_)
 - Remove deprecated Coverlib (`#369 <https://github.com/Komet/MediaElch/issues/369>`_)
 - Remove deprecated MediaPassion (`#449 <https://github.com/Komet/MediaElch/issues/449>`_)
 - Remove deprecated MovieMaze (`#386 <https://github.com/Komet/MediaElch/issues/386>`_)
 - Support KDE Breeze Dark theme (`#407 <https://github.com/Komet/MediaElch/issues/407>`_)
 - Scrape TV show tags when using IMDb
 - Detect duplicate movies
 - Create subdirectories
